# Report Type Outlines — NSO Nepal

Standard outlines for all 8 report types and the NSO Nepal Press Release template.
Load this file in Stage 3 when proposing an outline.

---

## 1. Statistical Bulletin / Release

**Purpose:** Regularly publish key statistical indicators (CPI, GDP, trade, employment)
**Audience:** Government ministries, media, general public
**Typical length:** 10–30 pages

**Standard Outline:**
```
1. Cover Page
2. Key Highlights Box (3–5 bullet points, most important numbers)
3. Introduction / Background
4. [Main Indicator Section(s)] — one section per indicator cluster
   e.g., 4.1 Overall Index / 4.2 Sub-category / 4.3 Regional Breakdown
5. Trend Analysis (comparison with previous periods)
6. Methodology Note (short — 1 paragraph)
7. Statistical Data Tables
8. Contact Information
```

---

## 2. Press Release

**Purpose:** Announce a finding or publication quickly, for media use
**Audience:** Journalists, general public
**Typical length:** 1–2 pages only

**Standard Outline:**
```
[NSO NEPAL LETTERHEAD / LOGO]
FOR IMMEDIATE RELEASE                          Date: [Baisakh DD, 2082 / DD Month 2025]

HEADLINE: [One factual, newsworthy sentence — the most important number]
(Subtitle if needed: one sentence of context)

KATHMANDU — [Opening paragraph: the single most important fact, in plain language.
             Answer: What happened? When? How much?]

[Body paragraph 1: Second most important finding. Compare to previous period.]

[Body paragraph 2: What is driving the change? Any regional or category breakdown.]

[Quote: "..." said [Full Name], Director General, NSO Nepal.]

[Boilerplate: One standard paragraph about NSO Nepal — who we are, what we do,
              our website URL.]

For more information, contact:
[Name] | [Title] | [Email] | [Phone]

###
(Three hash marks signal the end of a press release)
```

**Press Release Rules:**
- Most important information goes FIRST — journalists cut from the bottom
- One or two headline numbers only — not the full dataset
- No jargon — define any technical term in brackets
- Round numbers intelligently in the headline ("about 1.2 million" not "1,213,447")
- Must be issued simultaneously with the main report, not days later

## 3. Statistical Abstract / Compendium

**Purpose:** Comprehensive collection of data across sectors
**Audience:** Researchers, planners, international partners
**Typical length:** 100–300+ pages

**Standard Outline:**
```
1. Cover Page
2. Foreword (Director General)
3. Preface / About This Publication
4. Table of Contents
5. List of Tables and Figures
6. Abbreviations and Acronyms
7. Introduction
   7.1 Background and Scope
   7.2 Data Sources
   7.3 Notes on Methodology and Coverage
8. [Thematic Chapters — one per sector]
   e.g., Chapter 1: Population and Demographics
         Chapter 2: Education
         Chapter 3: Health
         Chapter 4: Economy and Employment
         Chapter 5: Agriculture
         (etc. — adapt as needed)
9. Summary Tables (cross-sector overview)
10. References and Data Sources
11. Annexes (metadata, definitions, classification systems)
```

---

## 4. Analytical / Thematic Report

**Purpose:** In-depth analysis of a specific topic or policy issue
**Audience:** Policymakers, donors, researchers, senior officials
**Typical length:** 40–120 pages

**Standard Outline:**
```
1. Cover Page
2. Foreword
3. Acknowledgements
4. Executive Summary (1–2 pages — write this LAST)
5. Table of Contents
6. List of Tables and Figures
7. Abbreviations
8. Introduction
   8.1 Background and Rationale
   8.2 Objectives of the Report
   8.3 Scope, Coverage, and Reference Period
9. Data Sources and Methodology
   9.1 Data Sources Used
   9.2 Analytical Framework
   9.3 Limitations and Caveats
10. [Thematic Findings — one chapter per theme]
    e.g., Chapter 3: National-level trends
          Chapter 4: Disaggregated analysis (by gender, region, age group)
          Chapter 5: Comparative analysis (trend over time / international benchmark)
11. Conclusions
12. Recommendations (if the brief calls for them)
13. References / Bibliography
14. Annexes
```

## 5. Census / Survey Report

**Purpose:** Document the full methodology, findings, and data tables from a census or major survey
**Audience:** All stakeholders — government, researchers, international organisations, public
**Typical length:** 100–500 pages

**Standard Outline:**
```
1. Cover Page
2. Foreword (Director General and/or Minister)
3. Preface
4. Acknowledgements
5. Table of Contents
6. List of Tables, Figures, and Maps
7. Abbreviations and Acronyms
8. Glossary of Key Terms
9. Executive Summary (write LAST)
10. Chapter 1: Introduction
    1.1 Background
    1.2 Objectives
    1.3 Legal and Institutional Framework
11. Chapter 2: Methodology
    2.1 Survey / Census Design
    2.2 Sampling Frame and Sample Size (for surveys)
    2.3 Questionnaire Design and Piloting
    2.4 Data Collection Process
    2.5 Data Processing, Editing, and Validation
    2.6 Weighting and Estimation (for surveys)
    2.7 Data Quality and Limitations
12. [Findings Chapters — one chapter per major theme]
    (structure varies by survey type)
13. Conclusions
14. References
15. Annexes
    - Full data tables
    - Survey questionnaire
    - Sampling frame details
    - Field operation notes
```

**Note:** For census and major survey reports, check whether UNFPA, UNICEF, or the UN Statistics Division has a template. Always use it if one exists.

## 6. Administrative / Internal Report

**Purpose:** Progress updates, evaluations, briefing notes — for internal use
**Audience:** NSO management, line ministries, donor coordinators
**Typical length:** 2–15 pages

**Standard Outline:**
```
1. Header Block: To / From / Date / Subject / Reference
2. Purpose / Background (1 short paragraph)
3. Current Status / Progress
   - What has been completed
   - What is in progress
   - What is pending or delayed (and why)
4. Key Issues / Challenges (if any)
5. Actions Required / Recommendations
6. Next Steps and Timeline
7. Annexes (if needed: data, organogram, budget table)
```

---

## 7. Technical / Methodology Report

**Purpose:** Explain in detail how data was collected, processed, and validated
**Audience:** Statisticians, technical peers, international reviewers
**Typical length:** 30–100 pages

**Standard Outline:**
```
1. Cover Page
2. Foreword / Preface
3. Table of Contents
4. List of Abbreviations
5. Executive Summary / Abstract
6. Introduction
   6.1 Background and Purpose of the Document
   6.2 Scope and Coverage
7. Survey / Data Collection Design
   7.1 Sampling Frame
   7.2 Sample Design and Size Determination
   7.3 Stratification and Clustering
8. Questionnaire and Data Collection Tools
   8.1 Questionnaire Design Process
   8.2 Pilot Testing
   8.3 Interviewer Training
9. Field Operations
   9.1 Enumeration Process
   9.2 Supervision and Quality Control in the Field
10. Data Processing
    10.1 Data Entry and Coding
    10.2 Editing and Imputation Rules
    10.3 Weighting Methodology
11. Data Quality Assessment
    11.1 Response Rates
    11.2 Non-response Bias Analysis
    11.3 Comparison with Administrative Sources
12. Limitations and Caveats
13. References
14. Annexes (formulas, code, questionnaire)
```

## 8. Policy Brief

**Purpose:** Concise, action-oriented summary for decision-makers
**Audience:** Ministers, senior government officials, donor heads
**Typical length:** 2–4 pages only

**Standard Outline:**
```
1. Title (clear and descriptive — state the issue)
2. Key Message (1 sentence — what decision-makers need to know)
3. Background (5–6 lines — why this issue matters now)
4. What the Evidence Shows (2–3 key findings, with numbers)
5. Policy Options (what could be done — present 2–3 realistic options)
6. Recommended Action (what NSO Nepal recommends, and why)
7. Further Reading / Data Sources
```

**Policy Brief Rules:**
- The recommendation must come CLEARLY — do not bury it
- Use bullet points and short paragraphs — a Minister must read this in 5 minutes
- Every claim must have a number behind it
- Avoid methodology detail — this is not the place for it

## Summary Decision Table

| Situation | Best Report Type |
|-----------|-----------------|
| Monthly/quarterly data ready (CPI, trade, employment) | Statistical Bulletin |
| Publishing a bulletin + want media coverage | Press Release (issued alongside) |
| Minister needs a 5-minute read before a meeting | Policy Brief |
| Completed a national census or survey | Census / Survey Report |
| Analysing trends across multiple indicators | Analytical / Thematic Report |
| Peer institution asks how data was collected | Technical / Methodology Report |
| Briefing director on project progress | Administrative / Internal Report |
| International donor wants country data summary | Statistical Abstract / Compendium |
